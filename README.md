# TCG Wiki

## Project Setup
```bash
yarn install
```

Then copy `.env.example` to `.env` and update the environment variables with your API key.

### Start the Development Server
```bash
yarn dev
```

### Build for Production
```bash
yarn build
```

### Preview the Production Build
```bash
yarn serve
```
